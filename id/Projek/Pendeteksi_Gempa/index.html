Apa itu Pendeteksi Gempa?

Pendeteksi Gempa adalah sebuah robot Discord yang dapat memberitau anda informasi gempa bumi, tsunami dan gunung api yang ada di seluruh Indonesia! Kami menggunakan data yang disediakan oleh BMKG dan Magma Indonesia. Jadi anda akan selalu up-to-date

Pertanyaan

Ini bukan bot resmi buatan BMKG dan Magma Indonesia! Robot ini tidak ada kaitan dengan keduanya. Kami menggunakan API BMKG dan scrapping website Magma API untuk mendapatkan informasi ini!

Ya, kamu bisa mengatur lokasi bencana sesuai dengan lokasimu.

Untuk menggunakan robot kami, pertama tama gunakan perintah /daftar lalu ikuti instruksi yang kami berikan. Setelah selesai mengikuti instruksi yang kami berikan, sekarang anda dapat menggunakan layanan kami.

Ada 5 buah data yang kami ambil untuk mengugnakan layanan ini, dan 5 data ini bersifat opsional. Jika kamu menggunakan /notifikasidm, maka ada 5 buah data yang di ambil, yaitu (lokasi), dan user ID discord. Jika kamu menggunakan /aktifkantsunami dan /aktifkangempa, maka ada 2 data yang kami ambil, yaitu server ID dan channel ID.

Untuk menghapus data anda, anda bisa menggunakan perintah /hapusakun. Perlu dicatat bahwa data yang di hapus tidak dapat dikembalikan

Hanya gabrielbjb saja yang bisa mengakses/melihat data anda.
