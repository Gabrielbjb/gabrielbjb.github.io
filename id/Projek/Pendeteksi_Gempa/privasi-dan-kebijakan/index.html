Kebijakan Penggunaan Bot

Sebelum menggunakan discord bot buatan kami, silahkan untuk membaca apa saja yang dapat kami lakukan pada data anda pada saat anda menggunakan robot kami. Data anda akan kami gunakan untuk mengirimkan pesan (gempa, volkano dan tsunami) ke server anda. Data yang kami ambil berupa Server ID (yang dimana berisi semua informasi profile server anda seperti nama server), Channel ID (berisi semua informasi pengaturan channel anda seperti nama channel), Owner ID (berisi semua informasi akun publik seperti nama anda), ID Pesan (hanya berisi informasi pesan yang kami kirim saja, kami tidak dapat melihat pesan yang ada di channel anda) yang menggunakan bot kami. Kami mungkin juga akan merekam aktivitas anda saat menggunakan perintah (/) di bot kami yang dimana akan kami gunakan untuk mendeteksi apakah anda manusia atau bukan. Kami berhak melarang anda menggunakan bot kami apabila anda ketahuan melanggar peraturan kami atau Term of services Discord
